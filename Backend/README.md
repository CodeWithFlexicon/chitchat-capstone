# Backend README

## Installation Guide

### Cloning the repository

```shell
git clone git@github.com:ericz02/chitchat-capstone.git
```

### Install packages

```shell
npm install
```

### Setup .env file

```js
DB_USER=
DB_HOST=
DB_NAME=
DB_PASSWORD=
DB_PORT=
SESSION_SECRET=
``` 


### Start the app

```shell
npm start
```

## Available commands

Running commands with npm `npm [command]`

| command         | description                              |
| :-------------- | :--------------------------------------- |
| `start`         | Starts a development instance of the app |
